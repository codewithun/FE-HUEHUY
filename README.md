# 🚀 FE Huhehuy Project (Next.js)

Frontend project berbasis Next.js untuk aplikasi Huhehuy.

---

## 📦 Requirements

Pastikan sudah install:

- **Node.js** (disarankan versi ≥ 18)
- **npm** atau **yarn**

Cek versi:

```bash
node -v
npm -v
```

---

## 📥 Clone Repository

Clone project dari repository:

```bash
git clone https://github.com/username/fe-huhehuy.git
```

Masuk ke folder project:

```bash
cd fe-huhehuy
```

---

## 📦 Install Dependencies

Install semua package yang dibutuhkan:

```bash
npm install
```

> Kalau kamu tipe yang suka buru-buru terus error, ini biasanya gara-gara step ini kelewat. Jadi jangan sok skip ya 😌

---

## ▶️ Menjalankan Project

Untuk menjalankan project di mode development:

```bash
npm run dev
```

Buka di browser:

```
http://localhost:3000
```
